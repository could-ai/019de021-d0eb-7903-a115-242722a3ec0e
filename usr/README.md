# تطبيق العداد (Compteur App)

هذا التطبيق هو تطبيق فلاتر (Flutter) بسيط يقوم بدور عداد رقمي، وهو مصمم ليعرض النصوص باللغتين الفرنسية والعربية معاً، بشكل أنيق وحديث.

## نظرة عامة
تم بناء هذا التطبيق كنموذج أولي لاستعراض كيفية عمل واجهات المستخدم في إطار عمل Flutter. يحتوي التطبيق على شاشة واحدة تعرض عدداً يتزايد كلما تم الضغط على الزر.

## قائمة الميزات
* **زيادة العداد**: زر عائم (Floating Action Button) يسمح بزيادة الرقم الحالي بمقدار واحد.
* **إعادة الضبط**: زر في شريط التطبيق (AppBar) يسمح بإعادة العداد إلى الصفر.
* **دعم لغوي**: النصوص معروضة باللغتين الفرنسية والعربية، مع ضبط اتجاه النص (RTL / LTR) ليتناسب مع اللغة.
* **تصميم عصري**: مبني على Material Design 3 بألوان مريحة للعين.

## الاستخدام المتوقع (User Flows)
1. يفتح المستخدم التطبيق فيرى العداد عند الرقم `0`.
2. يقرأ المستخدم التوجيهات بالفرنسية أو العربية.
3. يضغط المستخدم على زر الإضافة **(Ajouter / أضف)** في الأسفل، فيزيد الرقم.
4. إذا رغب المستخدم في تصفير العداد، يضغط على أيقونة التحديث **(Refresh)** أعلى الشاشة.

## التقنيات المستخدمة
* **إطار العمل**: Flutter
* **لغة البرمجة**: Dart
* **إدارة الحالة**: `setState` (StatefulWidget)

## تعليمات التشغيل والإعداد
1. تأكد من تثبيت بيئة عمل [Flutter](https://flutter.dev/docs/get-started/install).
2. قم بتنزيل أو استنساخ المستودع (Clone the repository).
3. افتح نافذة الأوامر (Terminal) في مجلد المشروع ونفذ:
   ```bash
   flutter pub get
   ```
4. لتشغيل التطبيق على الهاتف المحمول (أو المحاكي):
   ```bash
   flutter run
   ```
